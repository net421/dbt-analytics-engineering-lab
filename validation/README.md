# Generated Validation Evidence

Run `make verify` to regenerate:

- `dbt_build_summary.md`
- `portfolio_kpi_summary.csv`
- `dbt_artifacts/manifest.json`
- `dbt_artifacts/catalog.json`
- `dbt_artifacts/run_results.json`

GitHub Actions uploads the same files as the `dbt-validation-evidence` artifact for every pull request run.
